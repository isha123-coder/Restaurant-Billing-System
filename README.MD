This is a simple console-based Restaurant Billing System built using Python.
The program allows customers to order multiple items from a menu, enter quantity, and generates a formatted receipt with a grand total.


Features :

1.Display restaurant menu
2.Take multiple orders (infinite loop)
3.Quantity handling
4.Automatic bill calculation
5.Proper receipt format
6.User-friendly console interaction


Technologies Used :

Language: Python
Concepts:
Dictionary
While loop
Conditional statements
String formatting
User input handling
